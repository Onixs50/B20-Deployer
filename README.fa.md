# B20 Forge

**B20 Forge** یه اپ وبه برای ساختن توکن خودت روی **Base** با استاندارد جدید
**B20** — بدون نوشتن سالیدیتی، بدون دیپلوی دستی کانترکت. اسم و نماد و عرضه
رو وارد کن، کیف‌پولت رو وصل کن، سایت با یک تراکنش توکن رو برات می‌سازه.

## B20 چیه؟

B20 استاندارد توکن بومی خود Baseه: به‌جای اینکه یه قرارداد ERC-20 مخصوص خودت
دیپلوی کنی، توکن از طریق یه فکتوری داخلی که توی خود چین وجود داره ساخته
می‌شه. این باعث می‌شه ساخت توکن ارزون‌تر باشه، و هر توکن یه‌سری قابلیت ثابت
و آماده داره — mint، burn، pause، دسترسی‌های نقش‌محور — بدون اینکه کسی لازم
باشه اون منطق رو خودش بنویسه یا آدیت کنه.

## این سایت چیکار می‌کنه

- **ساخت توکن** — اسم، نماد، تعداد رقم اعشار و (اختیاری) سقف عرضه و مقدار
  mint اولیه رو وارد کن، کیف‌پولت رو وصل کن، دیپلوی بزن. تمام انکودینگ‌های
  روی زنجیره رو خود سایت انجام می‌ده.
- **مدیریت توکن بعد از دیپلوی** — هر توکنی که بسازی، یه پنل مدیریت داره که
  توش (به‌عنوان ادمین توکن) می‌تونی:
  - **Mint** کنی — توکن جدید برای هر آدرسی
  - **Burn** کنی — از موجودی خودت
  - **ارسال** کنی — ترنسفر ساده به یک آدرس
  - **ارسال گروهی** — mint یا ترنسفر به چند آدرس با هم، توی یک تراکنش (نه
    یک امضا برای هر گیرنده)
  - **Pause/Unpause** کنی — توقف موقت همهٔ ترنسفرها
  - **نقش بدی** (mint/burn/pause) به آدرس‌های دیگه
- **فقط روی Base Mainnet و Base Sepolia (تست‌نت)** — سایت قبل از اجازه‌دادن
  به هر امضایی، چک می‌کنه کیف‌پولت روی یکی از این دو شبکه باشه.
- **دوزبانه و RTL** — بین فارسی و انگلیسی سوییچ کن؛ جهت چیدمان خودکار عوض
  می‌شه.
- **تاریخچهٔ محلی توکن‌ها** — لیستی از توکن‌هایی که از همین مرورگر دیپلوی
  کردی، هرکدوم با یه کلیک بره توی پنل مدیریتش.

## این سایت چی *نیست*

- پول یا کلید خصوصی کسی رو نگه نمی‌داره. هر عملیات یه تراکنش عادی کیف‌پوله
  که خودت می‌بینی و امضا می‌کنی (MetaMask، Coinbase Wallet و... از طریق
  [Reown AppKit](https://reown.com)).
- محصول رسمی Base نیست — یه ابزار مستقله که روی استاندارد عمومی B20 و
  کانترکت‌های عمومی Base ساخته شده.

## پشتهٔ فنی

| لایه | انتخاب |
|---|---|
| فریم‌ورک | React + TypeScript + Vite |
| استایل | Tailwind CSS |
| کیف‌پول / زنجیره | wagmi، viem، Reown AppKit |
| انیمیشن | Framer Motion |
| کانترکت‌ها | سالیدیتی، دیپلوی‌شده روی Base (پوشهٔ [`/contracts`](./contracts)) |
| هاست عکس | IPFS از طریق Pinata (برای لوگوی توکن) |
